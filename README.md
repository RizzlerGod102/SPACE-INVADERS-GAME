# Space Invaders JS 🚀👾

## Description
A classic Space Invaders clone built with **JavaScript, CSS, and HTML**.  
Control your ship, shoot aliens, and try to get the highest score! 🏆

---

## How to Play 🎮
- **Move Left:** ⬅️ Arrow Left  
- **Move Right:** ➡️ Arrow Right  
- **Shoot:** 🔫 Space Bar

**Rules:**  
1. Destroy all aliens to increase the level. 💥  
2. Each alien destroyed gives you **100 points** ✨  
3. If an alien reaches your ship, the game is **over** ☠️  
4. Bullets disappear when they hit an alien or leave the screen 💨
5. 🚀 Refresh the browser to play again after Game Over

---

## Download & Play 💾
1. Go to the **GitHub repository** for this game.  
2. Click the **`Code`** button at the top right of the repo.  
3. Select **`Download ZIP`**.  
4. Extract the ZIP file to a folder on your computer.  
5. Open the `index.html` file in your **web browser** 🌐 to play.  
6. Enjoy shooting aliens! 👾🚀
